Download this file,
Rename it to "Zypherix.py",

cd ~/downloads
then run
python3 Zypheris.py

[ONLY WORKS ON KALI LINUX (OR ANY VM WITH KALI / LINUX]
-- Note: PY should be preinstalled.

To convert to exe run:

pyinstaller --onefile zypheris.py
